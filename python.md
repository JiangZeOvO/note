# Python 相关知识笔记

## 1. 概述

### 1.1 语言特性

Python 是一种面向对象编程,解释型,强类型的动态脚本语言,具有跨平台兼容性好等特点

**解释型** 通过解释器将编写的代码解析成机器码,仅在执行到指定代码时进行动态解析该行代码,这种特性使得 Python 跨平台兼容性好

**编译型** 与之相对的编译型语言通过编译器将编写的代码编译成机器码,在执行前将全部代码一次性编译完毕,一般情况下速度快过解释型语言,不过目前部分解释型语言的动态优化过于完善,以至于某些解释型语言性能超过编译型语言

### 1.2 安装与命令

**安装** 可以通过[Python 官方网站](https://www.python.org/)下载 Python 解释器进行安装

| 命令                     | 参数             | 含义             | 注释              |
|------------------------|----------------|----------------|-----------------|
| `python`               |                | 进入 Python 交互模式 |                 |
| `exit()`               |                | 退出 Python 交互模式 | 也可使用 `Ctrl + Z` |
| `where python`         |                | 查询 Python 安装路径 |                 |
| `pip install [module]` | `[module]` 为库名 | 下载第三方 Python 库 |                 |

复数的定义方式为 $a + bi$ ,其中 $a$ 为实部, $b$ 为虚部, $i$ 为虚数单位(在 Python 中用 $j$ 表示)

## 3.数据结构

### 3.1 序列

##### 注释

可迭代(可遍历),可排序,可重复

多种类型的统称,包括`str` `list` `tuple` `range` `bytes`

##### 用法

```python
# 索引
    var = "123456"
    print(var[0])
        # 输出"1",正数第一个下标的数据
    print(var[-1])
        # 输出"6",倒数第一个下标的数据
# 切片
#   不会修改源序列,会返回一个新的序列
#   var[start:stop:step]
#   [start] 起点 默认[0]
#   [stop] 终点 默认[len(this)]
#   [strp] 步长 默认[1]
#   截取数据包含[start]不包含[stop]
    var = "123456"
    print(var[2:5])
        # 输出"345"
    print(var[:5])
        # 输出"12345"
    print(var[5])
        # 输出"6"
    print(var[:])
        # 输出"123456"
    print(var[::2])
        # 输出"135"
    print(var[::-1])
        # 输出"654321" 反转数据
# 重载
#   不会修改源序列,会返回一个新的序列
    var = "123456"
    print(var * 2)
        # 输出"123456123456"
    print(var + "789")
        # 输出"123456789"
# 判断
#   i in 序列
#       [序列]中是否包含索引[i]
#   i not in 序列
#       [序列]中是否不包含索引[i]
# 遍历
    var = "123456"
    for i in var :
        pass
```

`round(2.5)` 结果是 `2`,而不是 `3` 在 Python 中,对于小数为 `.5` 的情况,它会四舍五入到最接近的偶数

`round(3.5)` 结果是 `4`,因为 `3.5` 四舍五入到最接近的偶数是 `4`

`round(2.675, 2)` 结果是 `2.67`,而不是 `2.68`,因为 `2.675` 在二进制中无法精确表示,实际存储的值略小于 `2.675`
,因此四舍五入结果为 `2.67`

### 13.3 结构

#### 树结构
包含`根节点`和`叶节点`

大于等于1个有限结点,组成的具有层次关系的集合

有且仅有1个节点没有[父节点],该节点称为树的`根节点`

除根外其余所有节点都有且只有1个`父节点`

每个节点都有一个以他为`根节点`的树
